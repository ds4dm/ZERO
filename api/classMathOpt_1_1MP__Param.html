class MathOpt::MP_Paramïƒ

This class handles parameterized mathematical programs (MP) Their form is the one of.

\[ \min_y \frac{1}{2}y^TQy + c^Ty + (Cx)^T y \]
Subject to
\[\begin{split}\begin{eqnarray} Ax + By &\leq& b \\ y &\geq& 0 \end{eqnarray}\end{split}\]

Subclassed by MathOpt::IP_Param, MathOpt::QP_Param

void detectBounds()ïƒ

Detects explicit bounds stated in the MP_Param formulation, and stores them implicitly. This is useful when the formulation of the parametrized mathematical program is processed through other steps (e.g., KKT conditions, LCPs, etcâ€¦) since any bound constraint can possibly slow down the final problem solving.
