const void makePlayerQP(unsigned int i)

Makes the Game::QP_Param corresponding to the i-th country.

  • First gets the Game::LCP object from Game::EPEC::PlayersLowerLevels and makes a Game::QP_Param with this LCP as the lower level

  • This is achieved by calling LCP::makeQP and using the objective value object in Game::EPEC::LeaderObjective

  • Finally the locations are updated owing to the complete convex hull calculated during the call to LCP::makeQP

    Note

    Overloaded as Models::EPEC::makePlayerQP()
